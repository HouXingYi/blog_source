# 第一周 基本概念

## 什么是数据结构

解决问题方法的效率和数据的组织方式有关，和空间的利用效率有关，也和算法的巧妙程度有关。

组织方式：

1. 数据对象集的逻辑结构。线性，树状，图状。

2. 数据对象集在计算机中的物理存储结构。数组存储，链表存储。

抽象数据类型（类型名称，数据对象集，操作集）

## 什么是算法

算法空间复杂度S(n)，时间复杂度T(n)。

时间复杂度计算方法：寻找执行次数t与问题的规模n的表达式，得出t>f(n)时算法退出，f(n)即为时间复杂度。意义为最深的执行次数（频度）与问题的规模n的关系。

## 应用实例：最大子列和问题

1. 直接穷举O(n^3)

2. 部分存储中间值的穷举O(n^2)

3. 分而治之O(NlogN)

4. 在线处理O(N)

思路：在线处理，计算当前子列是否对后续子列有正贡献（是否为正），若无正贡献，则抛弃。书P17。

```
#include<cstdio>

int main () {
	
	int n;
	scanf("%d", &n);
	int sum = 0, max = 0;
	
	for (int i = 0; i < n; i++) {
		
		int num;
		scanf("%d", &num);
	
		sum += num;
			
		// 更新最大子列和 
		if (sum > max) {
			max = sum;
		}
		
		// 无正贡献，抛弃 
		if (sum < 0) {
			sum = 0;
		}
		
	}
	
	printf("%d", max);
	return 0;
}
```

2. Maximum Subsequence Sum：是2004年浙江大学计算机专业考研复试真题，要求略高，选做。其实也不难，是本次课最后讲到的算法的改造，挑战一下吧~

思路：是上一题最大子列和的改编。新增加的要求为，记录最大子列和的头位置与尾位置。我的策略为在在线处理算法中，在更新最大子列和的时候也更新下头和尾的信息即可。注意全为负数的情况。

```
#include<cstdio>

int main () {
	
	int n;
	
	int list[10005];
	scanf("%d", &n);
	int sum = 0, max = -1;
	
	int isAllNegative = true;
	
	int tempFirst = 0; 
	int first = 0, end = 0;
	int count = 0;
	
	for (int i = 0; i < n; i++) {
		
		int num;
		scanf("%d", &num);
		
		// 将数字记录在list中。 
		list[i] = num;
		
		if (num >= 0) {
			isAllNegative = false;
		} 
		
		sum += num;
		
		// 更新最大值 
		if (sum > max) {
			max = sum;
			// 更新最大值时，确定最大值所绑定的起点与终点。 
			first = tempFirst; 
			end = i;
		}
		
		// 无正贡献，清零 
		if (sum < 0) {
			sum = 0;
			tempFirst = i + 1; // 清零时，将新的起点，设为下一位。 
		}
		
	}
	
	// 若全为负，则最大子列和为0，起点为列首，终点为列尾。 
	if (isAllNegative) {
		printf("%d %d %d", 0, list[0], list[n - 1]);
	} else {
		printf("%d %d %d", max, list[first], list[end]);	
	}
	
	return 0; 
}
```

```
List Merge( List L1, List L2) {
    List L3 = (List)malloc(sizeof(struct Node));
    List L = L3;
    List p = L1->Next;
    List q = L2->Next;
    while(p && q) {
        if(p -> Data < q -> Data) {
            L -> Next = p;
            p = p -> Next;
        } else {
            L -> Next = q;
            q = q -> Next;
        }
        L = L -> Next;
    }
    L -> Next = NULL;
    if(p) L -> Next = p;
    if(q) L -> Next = q;
    L1 -> Next = NULL;
    L2 -> Next = NULL;
    return L3;
}
```

2. 一元多项式的乘法与加法运算 在“小白专场”里，我们会详细讨论C语言实现的方法。对于不会C语言而不能做第1题的同学，本题一定要做；

思路：乘法，指数相加，系数相乘。加法，指数不变，系数相加。存储结构，采用数组分别存储。

```
#include <stdio.h>
#include <stdlib.h>

#define N 10000

int main() {
	
    int a[N] = { 0 };  // 第一个多项式 
    int b[N] = { 0 };  // 第二个多项式 
    
    int c[N] = { 0 };  // 求积
    int d[N] = { 0 };  // 求和
    
    int i, j;
    int x, z; //系数  指数
    int cnt = 0;

    scanf("%d", &i);  //读入第一个单项式
    while( i-- ) {
        scanf("%d %d", &x, &z);
        a[z] += x;   //指数为z的系数放在z位置上
    }

    scanf("%d", &i);  //读入第二个单项式
    while( i-- ) {
        scanf("%d %d", &x, &z);
        b[z] += x;
    }

    //多项式乘法
    for( i = N - 1; i >= 0; i--) {

        if(a[i]) {
            for( j = 0; j < N; j++) {
                if(b[j]) {
                    c[i + j] += a[i] * b[j]; // 指数相加，系数相乘 
                }
            }
        }
    }

    //输出多项式乘法结果
    for(i = N - 1; i >= 0; i--){
        if(c[i]) { // 不为0 
            if( cnt ){
                //处理结尾不能有多余空格
                printf(" ");
            }
            printf("%d %d", c[i], i);
            cnt++;
        }
    }
    if( !cnt ){
        printf("0 0");
    }
    printf("\n");

    //多项式加法
    for( i = N-1; i >= 0; i--){
        if(a[i]){
            d[i] += a[i]; // 指数不变，系数相加 
        }
    }
    for( j = N - 1; j >= 0; j--){
        if(b[j]){
    		d[j] += b[j]; // 指数不变，系数相加 
        }
    }

    //输出多项式加法结果
    cnt = 0;
    for( i=N-1; i>=0; i--){
        if( d[i] ){ // 不为0 
            if( cnt ){
                printf(" ");
            }
            printf("%d %d",d[i],i);
            cnt++;
        }
    }
    if(!cnt){
        printf("0 0");
    }

    return 0;
}
```

3. Reversing Linked List 根据某大公司笔试题改编的2014年春季PAT真题，不难，可以尝试；

思路：

采用柳神的做法，特点为：

* 将链表存储在双数组中，可不依赖next进行遍历，兼具链表和数组的特点。

* 翻转方式为：周期 * k  + 当前周期翻转后的位置

```
#include <iostream>
using namespace std;
int main() {
	
    int first, k, n, sum = 0;
    
    cin >> first >> n >> k; // 第一个结点地址，几个结点，翻转周期 
    
    int temp, 
	data[100005], // 存储值 
	next[100005], // 存储next指针
	list[100005], // 暂存 
	result[100005]; // 结果 
	
    for (int i = 0; i < n; i++) {
        cin >> temp; // 地址 
        cin >> data[temp] >> next[temp]; // 值，next地址 
    }
    
    // 遍历，从头到尾将链表放入list中。 
    while (first != -1) {
        list[sum++] = first;
        first = next[first];
    }
    
    for (int i = 0; i < sum; i++) result[i] = list[i]; // 将list复制给first
	 
	// 将list翻转后赋给result 
    for (int i = 0; i < (sum - sum % k); i++) {
    	result[i] = list[i / k * k + k - 1 - i % k]; // 周期 * k  + 当前周期翻转后的位置 
	}
    
    // 打印结果 
    for (int i = 0; i < sum - 1; i++)
        printf("%05d %d %05d\n", result[i], data[result[i]], result[i + 1]);
    printf("%05d %d -1", result[sum - 1], data[result[sum - 1]]);
    return 0;
}
```

4. Pop Sequence  是2013年PAT春季考试真题，考察队堆栈的基本概念的掌握，应可以一试。

思路：栈模拟

出栈的模拟

```
#include <iostream>
#include <stack>
#include <vector>

using namespace std;

int main() {
 
    int m, n, k;
    
    scanf("%d %d %d", &m, &n, &k); // 栈的最大容量，序列的长度，需要被检查的序列的个数 
    for(int i = 0; i < k; i++) {
        bool flag = false;
        stack<int> s;
        vector<int> v(n + 1);
        for(int j = 1; j <= n; j++) {
			scanf("%d", &v[j]);
		}
        int current = 1;
        
        // 核心算法
        for(int j = 1; j <= n; j++) {
            s.push(j);
            if(s.size() > m) break; // 超过栈的最大容量 
            // 模拟可能的出栈情况 
            while(!s.empty() && s.top() == v[current]) {
                s.pop();
                current++;
            }
        }
        
        // 完成全程的模拟出栈，即为yes 
        if(current == n + 1) flag = true;
        if(flag) printf("YES\n");
        else printf("NO\n");
    }
    return 0;
}
```

```
#include<stdio.h> 

#define MaxTree 11
#define ElementType char
#define Tree int
#define Null -1

struct TreeNode {
    ElementType Element;//树结点的值，字符
    Tree Left; 
    Tree Right;
} T1[MaxTree], T2[MaxTree];

// 创建树，树的结构为结构数组，返回根结点 
Tree BuildTree (struct TreeNode T[]) {
    int N, i = -1;
    int check[11]; // 判别根 
    char cl, cr;
    scanf("%d", &N);
    getchar();
    
    // 初始化 
    for(i = 0; i < N; i++) {
		check[i] = 0;
	}
	
    if(!N) {
		return Null;
	}
	
	// 创建树，并标记根结点 
    for(i = 0; i < N; i++) {
        scanf("%c %c %c", &T[i].Element, &cl, &cr);
        getchar();
        
        // 左子树 
        if(cl != '-') {
            T[i].Left = cl - '0';
            check[T[i].Left] = 1;
        } else {
			T[i].Left = Null;
		} 
        
        // 右子树
        if(cr != '-') {
            T[i].Right = cr - '0';
            check[T[i].Right] = 1;
        } else {
			T[i].Right = Null;
		}
    }
    
    for (i = 0; i < N; i++) {
		if(!check[i]) {
			return i; // 没被check的即为root 
		} 
	} 
    
}

// 是否是同构对比 
int Isomorphic(Tree R1, Tree R2) {
	
	// 都是空，是同构 
    if(R1 == Null && R2 == Null) {
		return 1;
	} 
    
    // 有一边为空，不是同构
    if(R1 == Null && R2 != Null) {
		return 0;
	}
    if(R1 != Null && R2 == Null) {
		return 0;
	} 
    
    // 根不相同，不是同构
    if(T1[R1].Element != T2[R2].Element) {
    	return 0;	
	} 
    
    // 左子树都是空的 
    if(T1[R1].Left == Null && T2[R2].Left == Null) {
		return Isomorphic(T1[R1].Right, T2[R2].Right); // 一起转向对比右子树 
	}
    
    // 左边同时不空，并且相等
    if(T1[R1].Left != Null && T2[R2].Left != Null && T1[T1[R1].Left].Element == T2[T2[R2].Left].Element) {
		// 无需反转对比左右子树 
		return Isomorphic(T1[R1].Left, T2[R2].Left) && Isomorphic(T1[R1].Right, T2[R2].Right); 
	} else {
		// 需要反转对比左右子树 
		return Isomorphic(T1[R1].Right, T2[R2].Left) && Isomorphic(T1[R1].Left, T2[R2].Right);
	} 
    
}

int main() {
	
    Tree R1, R2;
    
    R1 = BuildTree(T1);
    
    R2 = BuildTree(T2);
    
    if (Isomorphic(R1,R2) ) {
		printf("Yes\n");
	} else {
		printf("No\n");
	}
	
    return 0;
}
```

3. Tree Traversals Again 是2014年秋季PAT甲级考试真题，稍微要动下脑筋，想通了其实程序很基础，建议尝试。

思路：

https://pintia.cn/problem-sets/1077214780527620096/problems/1106167947099181058

考点：

书本109页，由两种遍历序列确定二叉树。先序序列可确定根节点（确定后到中序中找根），中序序列可确定左右子树的范围（再到先序中找）。

另有考点，先中后遍历的递归与非递归形式。

stringstream用于格式的转化（类似于sscanf）。

想法：

    1. 从一棵树的非递归中序遍历中，可以得到一个先序遍历（push的顺序）和一个中序遍历（pop的顺序）。
    2. 接着可以由一个先序遍历和一个中序遍历，得到一棵树。
    3. 再接着把这棵树，递归的进行后续遍历。


```
#include <iostream>
#include <stdio.h>
#include <vector>
#include <string>
#include <sstream>
#include <stack>
 
using namespace std;
 
int N, cur = 0;
vector<int> preOrder;
vector<int> inOrder;
 
// 树结构 
typedef struct TreeNode *Node;
struct TreeNode{
    int num;
    Node left, right;
    TreeNode() {
        left = NULL;
        right = NULL;
    }
};
 
// 根据值找index（在中序中寻找左右结点范围） 
int findRootIndex(int rootNum) { 
    for(int i = 0; i < N; i++) {
        if(inOrder[i] == rootNum) {
            return i;
        }
    }
    return -1;
}

// 树的创建 (通过先序和中序确定一棵树)
Node CreateTree(int left, int right) {
    if(left > right) return NULL;
    int root = preOrder[cur]; // 先序确定根结点 
    cur++;
    int rootIndex = findRootIndex(root); // 中序确定左右结点的范围 
    Node T = new TreeNode();
    T->num = root;
    if(left != right) {
        T->left = CreateTree(left, rootIndex-1);
        T->right = CreateTree(rootIndex+1, right);
    }
    return T; 
}
 
bool firstOutPut = true;

void PostOrder(Node T) {
    if(!T) return;
    PostOrder(T -> left);
    PostOrder(T -> right);
    if(firstOutPut) {
        printf("%d", T -> num);
        firstOutPut = false;
    }else{
        printf(" %d", T -> num);
    }
}
 
 
int main() {
    stringstream ss; // 用于将string转化为int
    string Nstr;
    getline(cin, Nstr);
    ss << Nstr;
    ss >> N;
    ss.clear();
    string input;
    stack<int> stk;
    int value;
    
    // 通过中序遍历的非递归算法确定先序遍历序列和中序遍历序列 
    for(int i = 0; i < N * 2; i++) {
        getline(cin, input);
        if(input[1] == 'u') { // push
```

## 二叉搜索树

二叉搜索树BST（Binary Search Tree）

二叉搜索树是施加了一定约束的特殊二叉树。

***特点：左小右大（左子树比根小，右子树比根大）。

操作：（五种）

### Find（）

利用左小右大的性质，比结点大的找右子树，比结点小的找左子树，与结点相等则返回。

```
// 非递归find
Position find (ElementType X, BinTree BST) {
	
	while(BST) {
		if (X > BST -> Data) {
			BST = BST -> Right;
		} else if (X < BST -> Data) {
			BST = BST -> Left;
		} else { // X == BST -> Data
			return BST;			
		}
	}
	
	return NULL; // find fail
}
```

### FindMin（）

最左端点，一直向左找，非递归版本

```
Position FindMin (BinTree BST) {
	if (BST) {

			BST = BST -> Left; 
		}
	}
	return BST;
}
```

### FindMax（）

最右端点，一直向右找，非递归版本


### Insert（）

插入之后还需要是二叉搜索树，与查找有些类似。找到需要插入地方，再一层一层的返回回来。

```
BinTree Insert (ElementType X, BinTree BST) {
	// 原树为空，为递归最深处，加入X结点，X结点的左右子树为空。 
	if (!BST) {
		BST = malloc(sizeof(struct TreeNode));
		BST -> Data = X;
		BST -> Left = BST -> Right = NULL; 
	} else if (X < BST -> Data) {
		BST -> Left = Insert(X, BST -> Left); // 向左走 
	} else if (X > BST -> Data) {
		BST -> Right = Insert(X, BST -> Right); // 向右走 
	} // else X === BST -> Data X 已经存在，什么都不做
	return BST;
}
```

### Delete（）

三种情况：

1. 要删除的是叶结点：直接删除

2. 要删除的只有一个儿子：将其父结点的指针指向要删除结点的孩子结点。（爷爷指向孙子，爸爸被删除了）

3. 要删除的结点有左右两个儿子：用另一结点替代被删除结点：右子树的最小元素或者左子树的最大元素。  

```
BinTree Delete (ElementType X, BinTree) {
	
	Position Tmp;
	
	if (X < BST -> Data) {
		BST -> Left = Delete(X, BST -> Left); // 向左走 
	} else if (X > BST -> Data) { // 向右走 
		BST -> Right = Delete(X, BST -> Right); // 向右走 
	} else { // 找到 
		
		if (BST -> Left && BST -> Right) { // 左右子树都有 
			Tmp = FindMin(BST -> Right); // 右子树最小的结点 
			BST -> Data = Tmp -> Data; // 右子树最小替代当前 
			BST -> Right = Delete(BST -> Data, BST -> Right); // 已替代，右子树最小的删除 
		} else { // 有一个或无子树
			Tmp = BST;
			if (!BST -> Left) {
				BST = BST -> Right; // 将右子树返回即可，即爷爷指向右子树 
			} else if(!BST -> Right) {
				BST = BST -> Left; // 将左子树返回即可，即爷爷指向左子树
			}
			free(Tmp);
		}
	}
	return BST;
}
```

```
void Union (setType S, setName Root1, setName Root2) {
	if (S[Root2] < S[Root1]) {
		S[Root2] += S[Root1];
		S[Root1] = Root2;
	} else {
		S[Root1] += S[Root2];
		S[Root2] = Root1;
```

## 什么是图

表示多对多的关系。

包含：
    一组顶点：通常用V表示顶点集合。
    一组边：通常用E表示边的集合。
        （V，W）无向边
        <V，W>有向边

抽象数据类型定义：
    类型名称：图
    数据对象集：G（V，E）由一个非空的有限顶点集合v和一个有限边集合E组成。
    操作集：。。。

如何在程序中表示一个图：
    1. 邻接矩阵：1表示有连接，0表示无连接（利用二重矩阵（或者一维数组）表示结点之间的关系）：这种方式用于密集的存储比较合算。
        邻接矩阵-有什么好处
        直观，简单，好理解
        方便检查任意一对顶点间是否存在边
        方便找任一顶点的所有“邻接点”
        方便计算任一顶点的度
    2. 邻接表：G[N]为指针数组，对应矩阵每行一个链表，只存非零元素。（数组的每一项是一个指针，指针指向当前这个结点所相邻的结点）：这种方式比较适合用于稀疏图的存储。
        方便找任一顶点的所有“邻接点”
        节约稀疏图的空间


## 图的遍历

深度优先搜索（Depth First Search, DFS）
    类似于树的先序遍历，利用堆栈。

广度优先搜索（Breadth First Search, BFS）
    类似于树的层序遍历，利用队列。

连通集：独立的图
